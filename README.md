# repo-identification
Repo prefix identification

Starting with:
Language, Library or framework prefix followed with the name of the project. 

Like this:
LA-todo = Laravel Framework and the name of the Project (todo)

*Prefixes:*	Language, Library or Framework 
AS	Astro JS   
AL	Alpine JS   
CX	Codux   
DJ	Django   
EX	Express JS   
JA	Java   
JS	JavaScript   
LA	Laravel   
LR	Laravel and React JS   
LV	Laravel and Vue JS   
NX	Next JS   
OC	Objective C   
PH	Php   
PY	Python   
RD	Redux JS   
RE	React JS   
ST	Strapi JS   
SV	Svelte JS   
SW	Swift   
TK	Tkinter   
VU	Vue JS   
XC	XCode   
XT	Xata
